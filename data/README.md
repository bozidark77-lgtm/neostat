# data/

Drop your real CSV/XLSX input files here (supplier `IZVEŠTAJ DOBAVLJAČA` and
`AMS Team … BREZA` exports).

Everything in this folder **except this README is git-ignored** (see
`../.gitignore`), because BREZA gate logs and supplier reports contain personal
data — names and card IDs. The app does not require files to live here; you can
pick them from anywhere in the GUI. This is just a convenient, safe-by-default
place to keep them next to the project.
